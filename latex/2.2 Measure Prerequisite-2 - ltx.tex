* Measurable function.

\begin{definition}
\label{def:measurable_function}
If $(X, \mathcal{M})$ and $(Y, \mathcal{N})$ are measurable spaces a mapping $f: X \rightarrow Y$ is called $(\mathcal{M}, \mathcal{N})$-measurable or measurable if $f^{-1}(E) \in \mathcal{M}$ for all $E \in \mathcal{N}$.
\end{definition}

\begin{remark}
\label{rem:composition_measurable}
Composition of measurable mappings are measurable.
\end{remark}

\begin{proposition}
\label{prop:generated_sigma_algebra}
If $\mathcal{N}$ is generated by $\mathcal{E}$, then $f: X \rightarrow Y$ is $(\mathcal{M}, \mathcal{N})$-measurable iff $f^{-1}(E) \in \mathcal{M}$ for all $E \in \mathcal{E}$.
\end{proposition}

\begin{proof}
\label{proof:generated_sigma_algebra}
Only if condition is trivial. For if condition, observe $\mathcal{F} = \{E \subseteq Y \mid f^{-1}(E) \in \mathcal{M}\}$ is a $\sigma$-algebra, that contains $\mathcal{E}$ by hypothesis; it therefore contains $\mathcal{N}$.
\end{proof}
